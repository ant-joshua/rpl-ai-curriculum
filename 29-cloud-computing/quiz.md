# Quiz: Cloud Computing

<div class="quiz">

**1. Manakah yang merupakan contoh SaaS (Software as a Service)?**

- [ ] Amazon EC2
- [ ] Microsoft Azure Virtual Machines
- [x] Google Workspace (Gmail, Docs)
- [ ] Heroku

**2. Serverless computing (misal AWS Lambda) berarti:**

- [ ] Aplikasi berjalan tanpa server sama sekali
- [x] Developer tidak perlu mengelola server; cloud provider menangani scaling otomatis
- [ ] Server hanya berjalan di malam hari
- [ ] Menggunakan dedicated physical server

**3. Kubernetes adalah platform untuk:**

- [ ] Menyimpan file statis di cloud
- [x] Orchestration container — deploy, scale, dan manage container otomatis
- [ ] Membuat virtual machine
- [ ] Monitoring cloud cost

**4. Infrastructure as Code (IaC) seperti Terraform memungkinkan:**

- [ ] Menulis kode aplikasi lebih cepat
- [x] Mendefinisikan dan mengelola infrastruktur cloud melalui file konfigurasi
- [ ] Mengganti bahasa pemrograman backend
- [ ] Mengetes API secara otomatis

**5. Perbedaan utama antara IaaS dan PaaS adalah:**

- [x] IaaS memberi kontrol atas OS dan middleware; PaaS mengelola runtime sehingga dev hanya fokus ke kode
- [ ] IaaS hanya untuk storage; PaaS untuk compute
- [ ] IaaS gratis; PaaS berbayar
- [ ] Tidak ada perbedaan

**6. Fitur auto-scaling di cloud berguna untuk:**

- [ ] Mengganti server yang rusak
- [x] Menambah/mengurangi jumlah instance secara otomatis berdasarkan beban trafik
- [ ] Membackup database setiap jam
- [ ] Enkripsi data secara real-time
