# 04. Web Basics (HTML/CSS/Tailwind)

> **Level:** 🌱 Beginner  
> **Jam:** 6 (3 minggu × 2 sesi)  
> **Prasyarat:** Modul 01 (JS Fundamentals)  
> **Output:** Landing page pribadi + Dashboard API live di Vercel  

## Tujuan Pembelajaran

Setelah modul ini, kamu bisa:
- Nulis HTML semantic dengan struktur header, nav, main, section, article, footer
- Layout pake CSS Flexbox & Grid
- Bikin responsive design tanpa framework
- Styling cepat pake Tailwind utility classes
- Implementasi dark mode pake Tailwind
- Manipulasi DOM pake JavaScript
- Fetch data dari API publik dan tampilin di halaman
- Deploy project ke Vercel
