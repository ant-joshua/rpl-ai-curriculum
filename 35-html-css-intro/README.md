## Tujuan Pembelajaran

Setelah modul ini, kamu bisa:

- Nulis struktur HTML sendiri dari nol
- Paham tag, atribut, semantic HTML
- Styling halaman pake CSS (warna, font, layout)
- Bikin layout responsive pake Flexbox & Grid
- Bikin & styling form HTML
- Deploy website statis ke Vercel/Netlify
