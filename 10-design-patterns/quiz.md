# Quiz: Design Patterns

> Jawab pertanyaan di bawah dengan klik pilihan yang benar.

<div class="quiz">

**1. SOLID adalah...**

- [ ] Bahasa pemrograman
- [x] Lima prinsip desain OOP
- [ ] Framework testing
- [ ] Database design

**2. S dalam SOLID (Single Responsibility) berarti...**

- [ ] Satu file untuk semua kode
- [x] Satu kelas punya satu alasan untuk berubah
- [ ] Satu fungsi untuk semua operasi
- [ ] Satu repository untuk semua branch

**3. Singleton Pattern memastikan...**

- [ ] Sebuah kelas bisa di-inherit berkali-kali
- [x] Hanya satu instance dari sebuah kelas yang dibuat
- [ ] Semua method bersifat statis
- [ ] Objek bisa di-clone

**4. Factory Pattern digunakan untuk...**

- [x] Membuat object tanpa expose logic instantiation ke client
- [ ] Menghancurkan object
- [ ] Menyimpan object ke database
- [ ] Mengurutkan array

**5. Observer Pattern memungkinkan...**

- [ ] Satu object memodifikasi object lain secara langsung
- [x] Satu object (subject) memberi tahu banyak observer saat terjadi perubahan
- [ ] Object berkomunikasi via HTTP
- [ ] Object menyimpan data ke file

**6. Huruf D dalam SOLID (Dependency Inversion) berarti...**

- [ ] Database harus terpusat
- [ ] Dependency harus dihapus
- [x] Abstraksi tidak boleh bergantung pada detail, detail bergantung pada abstraksi
- [ ] Semua method harus double

**7. Functional Programming (FP) menghindari...**

- [ ] Function
- [x] Side effects dan mutable state
- [ ] Parameter
- [ ] Return value

**8. `Array.map()` adalah contoh pattern functional yang disebut...**

- [ ] Loop biasa
- [x] Higher-order function
- [ ] Recursion
- [ ] Inheritance

**9. Prinsip Open/Closed (OCP) menyatakan kelas harus...**

- [ ] Terbuka untuk perubahan dan tertutup untuk ekstensi
- [ ] Terbuka untuk semua modifikasi
- [x] Tertutup untuk perubahan, terbuka untuk ekstensi
- [ ] Hanya bisa digunakan sekali

**10. Liskov Substitution (LSP) berarti...**

- [ ] Class turunan harus punya method tambahan
- [x] Object class turunan bisa menggantikan object class induk tanpa mengubah kebenaran program
- [ ] Semua method harus public
- [ ] Class induk tidak boleh punya method
