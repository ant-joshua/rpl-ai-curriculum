# Quiz: TypeScript

> Jawab pertanyaan di bawah dengan klik pilihan yang benar.

<div class="quiz">

**1. TypeScript adalah...**

- [ ] Bahasa pemrograman baru yang berbeda dari JavaScript
- [x] Superset JavaScript dengan type system statis
- [ ] Framework frontend seperti React
- [ ] Database management system

**2. Cara mendefinisikan tipe untuk properti `nama` yang hanya bisa diisi string?**

- [ ] `nama: string`
- [x] `nama: string`
- [ ] `nama :: string`
- [ ] `const nama = string`

**3. `interface` vs `type` — mana yang bisa di-extend (extends)?**

- [x] interface
- [ ] type
- [ ] Keduanya
- [ ] Tidak ada

**4. Generic function `<T>(arg: T): T` memungkinkan...**

- [ ] Fungsi hanya menerima tipe number
- [x] Fungsi bekerja dengan berbagai tipe tanpa kehilangan type safety
- [ ] Fungsi mengabaikan tipe parameter
- [ ] Fungsi hanya untuk array

**5. File `tsconfig.json` digunakan untuk...**

- [ ] Menulis kode TypeScript
- [x] Mengatur compiler options TypeScript
- [ ] Menyimpan dependency project
- [ ] Menjalankan aplikasi

**6. Apa tipe untuk nilai yang bisa `string` atau `null`?**

- [ ] `string`
- [ ] `any`
- [x] `string | null`
- [ ] `string?`

**7. Perbedaan `any` dan `unknown`?**

- [ ] Sama saja
- [x] `any` mengaktifkan semua operasi tanpa check, `unknown` butuh type narrowing
- [ ] `unknown` tidak ada di TypeScript
- [ ] `any` lebih aman

**8. `readonly` pada properti interface berarti...**

- [ ] Properti bisa diubah kapan saja
- [x] Properti hanya bisa dibaca, tidak bisa diubah setelah inisialisasi
- [ ] Properti opsional
- [ ] Properti hanya untuk debug

**9. Enum di TypeScript digunakan untuk...**

- [ ] Menulis angka desimal
- [x] Mendefinisikan kumpulan nilai konstanta yang bernama
- [ ] Mengimpor modul
- [ ] Membuat kelas abstrak

**10. Compile TypeScript ke JavaScript menggunakan perintah...**

- [ ] `node`
- [ ] `ts-node`
- [x] `tsc`
- [ ] `npm run`
