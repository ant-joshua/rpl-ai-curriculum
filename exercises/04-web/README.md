## Cara Menjalankan

1. Buka file `.md` di LMS.
2. Copy starter code HTML ke ExerciseRunner (exerciseType: 'html').
3. Lengkapi bagian yang perlu diisi (ditandai `// TODO` atau `/* ? */`).
4. Klik **Run** untuk melihat preview/output.
5. Untuk exercise CSS: resize browser untuk cek responsive.
