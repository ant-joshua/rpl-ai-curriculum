## Cara Menjalankan

1. Buka file `.md` di LMS.
2. Copy starter code dan simpan sebagai file `.js`.
3. Jalankan dengan Node.js: `node nama-file.js`.
4. Untuk Express server: jalankan dan test dengan curl atau browser.
5. Bandingkan output dengan expected output.
