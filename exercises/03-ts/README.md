## Cara Menjalankan

1. Buka file `.md` di LMS.
2. Copy starter code ke ExerciseRunner.
3. Lengkapi type annotations dan implementasi.
4. Klik **Run** untuk melihat output.
5. Jalankan test cases untuk verifikasi.
