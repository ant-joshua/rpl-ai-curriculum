## Cara Menjalankan

1. Buka file `.md` di LMS.
2. Copy perintah-perintah Git ke terminal.
3. Jalankan step by step sesuai instruksi.
4. Bandingkan output dengan expected output.
5. Kerjakan tugas tambahan jika ada.
