---

## 📋 Deskripsi

Modul **Portfolio Project Series** adalah puncak dari kurikulum RPL AI. Berbeda dari modul sebelumnya yang bersifat tutorial, modul ini berisi **5 proyek mandiri** yang dikerjakan step-by-step. Setiap proyek dirancang untuk menjadi portfolio piece yang berdiri sendiri.

Kamu akan membangun:
1. **Landing Page** — landing page personal portfolio dengan HTML/CSS/Tailwind, deploy ke Vercel/Netlify
2. **CRUD API** — Bookshelf API dengan Express + TypeScript + Prisma + PostgreSQL, deploy ke Railway
3. **Fullstack App** — Todo App dengan React + Express + JWT/OAuth, deploy penuh
4. **AI Agent App** — AI Chat Assistant dengan Mastra AI + Next.js
5. **Production Deploy** — Setup production-grade dengan Docker, VPS, CI/CD, monitoring

## 🎯 Tujuan Pembelajaran

Setelah menyelesaikan modul ini, kamu akan mampu:

- **Mendeploy** aplikasi web ke berbagai platform (Vercel, Netlify, Railway, VPS)
- **Membangun REST API** production-ready dengan validasi, error handling, dan dokumentasi
- **Mengimplementasikan autentikasi** JWT dan OAuth Google
- **Membuat AI agent** dengan tool calling dan RAG memory
- **Mengkonfigurasi production deployment** dengan Docker, SSL, CI/CD, dan monitoring
- **Menulis README** dan dokumentasi API yang profesional
- **Menampilkan 5 proyek real** di portfolio GitHub

## 📝 Cara Menggunakan Modul Ini

1. Baca README dan pahami deskripsi proyek
2. Ikuti step-by-step di setiap file sesi
3. Jangan copy-paste mentah — pahami setiap baris kode
4. Selesaikan latihan di akhir setiap sesi
5. Push ke GitHub setiap selesai satu milestone
6. Deploy dan dapatkan live URL

> **💡 Tip:** Satu repositori GitHub per project = 1 poin portfolio. 5 project = 5 poin. Lebih baik dari 1 repositori dengan 5 folder.
