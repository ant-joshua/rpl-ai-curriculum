# Quiz: Linux & Terminal

<div class="quiz">

**1. Perintah untuk melihat daftar file lengkap dengan permission, owner, dan ukuran:**

- [ ] `ls`
- [x] `ls -la`
- [ ] `find .`
- [ ] `cat`

**2. Operator pipe (`|`) di terminal digunakan untuk:**

- [ ] Menjalankan dua perintah secara paralel
- [x] Mengirim output satu perintah sebagai input ke perintah lain
- [ ] Membatalkan perintah yang sedang berjalan
- [ ] Membaca file konfigurasi

**3. Apa arti permission `-rwxr--r--`?**

- [ ] File, owner bisa baca/tulis, group dan other bisa execute
- [x] File, owner bisa baca/tulis/execute, group dan other hanya baca
- [ ] Direktori, semua user bisa full control
- [ ] File, semua user bisa baca/tulis

**4. Perintah untuk mencari proses yang berjalan dengan nama "nginx":**

- [ ] `find nginx`
- [x] `ps aux | grep nginx`
- [ ] `locate nginx`
- [ ] `list nginx`

**5. Manakah yang BUKAN shell built-in command?**

- [ ] `cd`
- [ ] `echo`
- [x] `grep`
- [ ] `export`

**6. Untuk mengubah izin file `script.sh` menjadi executable oleh owner, perintah yang tepat:**

- [ ] `chmod 644 script.sh`
- [x] `chmod +x script.sh`
- [ ] `chown u+x script.sh`
- [ ] `chmod 777 script.sh`
