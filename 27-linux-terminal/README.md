# 27. Linux Terminal & Shell Scripting

> **Level:** 🌱 Beginner  
> **Jam:** 8 (4 minggu × 2 sesi)  
> **Prasyarat:** Dasar komputer, bisa navigasi file  
> **Output:** Kumpulan shell script otomatisasi + monitoring

## Tujuan Pembelajaran

Setelah modul ini, kamu bisa:

- Navigasi & manipulasi file system Linux (cd, ls, cp, mv, rm, find, permissions)
- Manage proses & networking (ps, top, kill, pipes, redirects, curl, SSH, tmux)
- Text processing pake grep, sed, awk, cut, sort, uniq, jq
- Nulis bash script dari nol (variable, loop, if/else, function, error handling)
- Bikin cron job, alias, dan script otomatisasi sendiri
