# 📊 Rubrik Penilaian

## Tugas Mingguan (40%)

| Kriteria | 0 (Tidak) | 1 (Kurang) | 2 (Baik) | 3 (Sangat Baik) |
|----------|-----------|-------------|----------|------------------|
| Kode jalan | Error | Jalan dengan bug | Jalan semua | Jalan + edge cases |
| Git commit | Gak pake Git | 1 commit | Commit per fitur | Commit message rapi |
| AI Ethics | Copas tanpa paham | — | Prompt dicatat | Paham + improve kode |
| Code style | Berantakan | Rapi sedikit | Rapi + naming jelas | Rapi + komentar |

## Final Project (40%)

| Kriteria | Bobot | 0 | 1 | 2 | 3 |
|----------|-------|---|---|---|---|
| Functionality | 30% | Gak jalan | Jalan sebagian | Semua fitur | Plus extra |
| Frontend | 20% | Gak ada | HTML doang | Responsive | Polish |
| Backend + DB | 20% | Gak ada | API doang | API + DB | Plus auth |
| AI Feature | 15% | Gak ada | Pake AI API | Agent + tools | Agent + memory + RAG |
| Deploy | 10% | Local | Vercel/Railway | Both | Custom domain |
| Presentasi | 5% | Gak demo | Demo error | Demo lancar | + Code walkthrough |

## Partisipasi (20%)

- Kehadiran: 10%
- Aktif bertanya/jawab: 5%
- Bantu teman: 5%

## Passing Grade

| Grade | Nilai |
|-------|-------|
| A (Sangat Baik) | >= 85 |
| B (Baik) | >= 70 |
| C (Cukup) | >= 55 |
| D (Tidak Lulus) | < 55 |
