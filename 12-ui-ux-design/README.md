# 12. UI/UX Design — Bikin Aplikasi yang Gak Cuma Jalan, Tapi Juga Enak Dilihat

> **Level:** 🪴 Menengah  
> **Jam:** 8 (4 minggu × 2 sesi)  
> **Prasyarat:** Bootstrap/Tailwind atau CSS dasar  
> **Output:** Figma prototype interaktif (3 screen) + design system mini (5 komponen)

## Tujuan Pembelajaran

Setelah modul ini, kamu bisa:
- Nerapin design thinking dari empathize sampai test
- Bikin wireframe (low-fi → hi-fi) di Figma
- Pake color theory, typography, layout grid dengan benar
- Bikin interactive prototype dengan Figma
- Bangun design system mini (komponen reusable)
- Ngerapin accessibility (WCAG, ARIA, contrast ratio)
- Nyusun portfolio UI/UX yang menarik

## Output Akhir Modul

> **Figma prototype interaktif + design system mini** — aplikasi sederhana (todo app / catatan keuangan / jadwal pelajaran) dengan 3 screen (Login, Dashboard, Detail) + 5 komponen reusable (button, input, card, navbar, icon) + prototyping flow end-to-end.

## AI Prompt Exercises

Sepanjang modul, latihan pake AI:
- "Generate 3 user personas for [app idea] based on these pain points: ..."
- "Critique this wireframe layout — what breaks hierarchy and alignment?"
- "Suggest a color palette for [app type] with WCAG AA compliance"
- "Turn this HTML snippet into accessible semantic HTML with ARIA labels"
- "Generate a design system spec table for these components: button, input, card"
