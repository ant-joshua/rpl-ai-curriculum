# Quiz: Web Performance

<div class="quiz">

**1. Core Web Vitals mencakup tiga metrik utama yaitu:**

- [x] LCP (Largest Contentful Paint), FID (First Input Delay), CLS (Cumulative Layout Shift)
- [ ] TTFB, DOMContentLoaded, Load Event
- [ ] CPU Usage, Memory Usage, Network Speed
- [ ] Time to First Byte, Time to Interactive, Speed Index

**2. Lazy loading gambar bertujuan untuk:**

- [ ] Mempercantik tampilan website
- [x] Menunda loading gambar sampai gambar tersebut mendekati viewport
- [ ] Mengompres ukuran gambar otomatis
- [ ] Mengganti format gambar ke WebP

**3. Code splitting memungkinkan:**

- [ ] Menulis kode tanpa import
- [x] Memecah bundle JavaScript menjadi chunk yang dimuat sesuai kebutuhan (on-demand)
- [ ] Menggabungkan semua file CSS jadi satu
- [ ] Menghapus kode yang tidak terpakai secara otomatis

**4. Lighthouse adalah tools untuk:**

- [ ] Menulis CSS lebih cepat
- [x] Mengaudit performa, aksesibilitas, SEO, dan best practices sebuah halaman web
- [ ] Debugging JavaScript runtime
- [ ] Membuat animation web

**5. HTTP caching strategi yang memungkinkan browser menyimpan resource untuk waktu tertentu adalah:**

- [ ] ETag
- [x] Cache-Control: max-age
- [ ] Last-Modified
- [ ] Service Worker

**6. Apa dampak utama render-blocking resources terhadap performa?**

- [ ] Meningkatkan skor SEO
- [x] Menunda rendering halaman karena CSS/JS harus dimuat dan diproses dulu
- [ ] Menambah ukuran bundle
- [ ] Mempercepat Time to Interactive
