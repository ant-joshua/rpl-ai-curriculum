# Quiz: Soft Skills for Developer

<div class="quiz">

**1. Apa tujuan utama daily standup dalam Scrum?**

- [ ] Melaporkan progress ke atasan
- [x] Sinkronisasi tim, identifikasi blocker, dan rencana kerja hari itu
- [ ] Evaluasi kinerja anggota tim
- [ ] Membahas fitur baru untuk sprint berikutnya

**2. Teknik prioritas mana yang membagi tugas ke dalam kuadran Urgent–Important?**

- [ ] Pomodoro Technique
- [ ] Timeboxing
- [x] Eisenhower Matrix
- [ ] Scrum Poker

**3. Saat client minta fitur baru di luar scope yang sudah disepakati, langkah paling profesional adalah:**

- [ ] Langsung mengerjakan agar client senang
- [ ] Menolak mentah-mentah tanpa diskusi
- [x] Diskusikan dampak scope creep, ajukan change request, dan negosiasi deadline/biaya
- [ ] Diam-diam menambahkannya ke backlog

**4. Manakah yang BUKAN praktik komunikasi async yang efektif?**

- [ ] Dokumentasi keputusan di write-up
- [ ] Gunakan thread Slack/Discord agar konteks terjaga
- [x] Panggil meeting dadakan untuk setiap pertanyaan kecil
- [ ] Tulis pesan dengan konteks dan lampiran lengkap

**5. Growth mindset menurut Carol Dweck adalah:**

- [ ] Percaya bahwa bakat dan kecerdasan bersifat tetap
- [x] Percaya bahwa kemampuan bisa dikembangkan lewat usaha dan belajar
- [ ] Fokus pada hasil instan tanpa proses
- [ ] Menghindari tantangan untuk menjaga reputasi

**6. Apa penyebab utama impostor syndrome pada developer junior?**

- [ ] Kemampuan teknis yang benar-benar rendah
- [x] Merasa tidak pantas meskipun sudah punya bukti kompetensi
- [ ] Lingkungan kerja yang terlalu santai
- [ ] Kurangnya sertifikasi resmi
