**1. Apa kepanjangan dari STAR dalam metode behavioral interview?**

- [ ] Start, Task, Action, Result
- [x] Situation, Task, Action, Result
- [ ] Strategy, Time, Action, Review
- [ ] Skill, Task, Ability, Result

**2. Contoh jawaban yang menggunakan metode STAR adalah?**

- [ ] "Saya bisa coding"
- [x] "Saat proyek akhir (Situation), saya bertanggung jawab atas backend (Task), saya implementasi REST API dengan Express (Action), dan API bisa handle 500 request/detik (Result)"
- [ ] "Saya pernah kerja di startup"
- [ ] "Saya suka programming"

**3. Struktur data apa yang paling tepat untuk implementasi queue (FIFO)?**

- [ ] Stack
- [x] Queue (LinkedList)
- [ ] Array
- [ ] Tree

**4. Kompleksitas waktu pencarian data pada binary search tree (BST) yang balanced?**

- [ ] O(1)
- [x] O(log n)
- [ ] O(n)
- [ ] O(n²)

**5. Dalam system design interview, apa yang pertama kali harus dilakukan?**

- [x] Clarify requirements dan batasan sistem
- [ ] Langsung menggambar diagram database
- [ ] Memilih teknologi stack
- [ ] Menulis kode

**6. Database manakah yang lebih cocok untuk menyimpan data tidak terstruktur seperti JSON?**

- [ ] PostgreSQL
- [x] MongoDB (NoSQL)
- [ ] MySQL
- [ ] SQLite

**7. Yang WAJIB ada di portfolio GitHub untuk fresh graduate?**

- [ ] Sertifikat kursus
- [x] Project nyata dengan README jelas, live demo, dan kode rapi
- [ ] Foto profil
- [ ] 1000 contribution dalam seminggu

**8. Cara menjawab pertanyaan "Ceritakan tentang diri Anda" di interview?**

- [ ] Bacakan CV dari awal
- [x] Berikan elevator pitch: latar belakang, skill utama, pencapaian relevan, goals
- [ ] Cerita hobi
- [ ] Diam

**9. Apa itu whiteboard coding interview?**

- [ ] Interview menggambar diagram
- [x] Interview di mana kandidat menulis kode (di papan tulis atau editor) untuk menyelesaikan soal algorithm
- [ ] Interview presentasi slide
- [ ] Interview desain UI

**10. Pertanyaan terbaik untuk ditanyakan ke interviewer di akhir sesi?**

- [ ] "Berapa gaji?"
