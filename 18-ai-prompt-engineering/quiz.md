**1. Apa yang dimaksud dengan zero-shot prompting?**

- [ ] Memberi contoh output sebelum prompt
- [x] Meminta AI melakukan tugas tanpa memberikan contoh sama sekali
- [ ] Prompt yang gagal menghasilkan output
- [ ] Prompt dengan 0 karakter

**2. Teknik few-shot prompting berarti?**

- [ ] Prompt hanya boleh 5 kata
- [x] Memberikan beberapa contoh input-output sebelum pertanyaan utama
- [ ] Prompt untuk model dengan parameter sedikit
- [ ] Mengirim prompt berulang kali

**3. Apa tujuan dari Chain-of-Thought (CoT) prompting?**

- [ ] Membuat AI menjawab lebih cepat
- [x] Membuat AI menunjukkan langkah berpikir logis sebelum memberikan jawaban
- [ ] Mengurangi jumlah token yang digunakan
- [ ] Menggabungkan beberapa prompt jadi satu

**4. Teknik prompt yang membagi tugas kompleks menjadi langkah-langkah kecil?**

- [ ] Zero-shot
- [ ] Few-shot
- [x] Step-by-step / decomposition
- [ ] Role prompting

**5. Manakah contoh prompt yang PALING SPESIFIK dan efektif?**

- [ ] "Bantu saya coding"
- [x] "Buat fungsi TypeScript untuk validasi email dengan regex, sertakan error handling untuk input kosong dan format salah"
- [ ] "Buat aplikasi"
- [ ] "Tolong"

**6. Apa fungsi memberikan persona/role dalam prompt (role prompting)?**

- [ ] Memberi nama pada AI
- [x] Membantu AI mengadopsi perspektif tertentu sehingga output lebih sesuai konteks
- [ ] Membatasi kemampuan AI
- [ ] Mempercepat respon

**7. Parameter apa di API LLM yang mengontrol seberapa acak / kreatif output?**

- [ ] max_tokens
- [x] temperature
- [ ] top_p
- [ ] frequency_penalty

**8. Teknik prompt untuk memastikan output dalam format terstruktur (JSON, tabel, list)?**

- [ ] Zero-shot
- [ ] Chain-of-Thought
- [x] Format specification / structured output prompting
- [ ] Prompt chaining

**9. Apa yang dimaksud dengan prompt injection?**

- [ ] Prompt yang terlalu panjang
- [x] Serangan dengan menyisipkan instruksi jahat dalam input user untuk memanipulasi AI
- [ ] Prompt yang dijalankan berulang
- [ ] Teknik optimasi prompt

**10. Cara terbaik menangani prompt injection pada aplikasi AI chatbot?**

- [ ] Melarang semua input user
- [x] Memisahkan instruksi sistem dari input user, validasi input, dan batasi delimiter
- [ ] Menggunakan model yang lebih murah
