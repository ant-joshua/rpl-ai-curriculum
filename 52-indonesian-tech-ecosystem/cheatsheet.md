## Freelance

### Platform
| Lokal | Global |
|-------|--------|
| Sribu, Fastwork, Projects.co.id, Sayaplus | Upwork, Freelancer, Fiverr, Toptal |

### Rate Calculator
```
Rate Min = Target Pendapatan Bulanan / Jam Efektif (100 jam)
Rate Ideal = Rate Min × 1.3
Rate Premium = Rate Min × 2
```

### Benchmark Rate Indonesia (2024-25)
| Level | Web Dev | Mobile Dev | UI/UX |
|-------|---------|------------|-------|
| Junior (0-1th) | Rp 50-100K/jam | Rp 60-120K/jam | Rp 40-80K/jam |
| Mid (1-3th) | Rp 100-200K/jam | Rp 120-250K/jam | Rp 80-150K/jam |
| Senior (3+th) | Rp 200-400K/jam | Rp 250-500K/jam | Rp 150-300K/jam |

### Client Communication Flow
1. **Briefing** — tanya tujuan, target, referensi, deadline, budget, deliverable
2. **Proposal** — scope, excluded, timeline, harga, revisi, payment (DP 50%)
3. **Pengerjaan** — update tiap 2-3 hari, konfirmasi sebelum lanjut
4. **Revisi** — minor gratis, mayor di luar scope, batasi jumlah revisi
5. **Payment** — escrow atau DP 30-50% + pelunasan

### Pajak Freelancer
- **NPWP** — wajib, daftar di ereg.pajak.go.id
- **PPh Final (PP 23)** — 0.5% omzet < Rp 4.8M/thn
- **PPh 23** — 2% dipotong klien badan usaha
- **Invoice** — nama, NPWP, detail pekerjaan, total, rekening

## Job Search

### CV ATS-Friendly
| ✅ Lolos | ❌ Gagal |
|----------|----------|
| PDF text-based / DOCX | JPG/PNG, hasil scan |
| 1 kolom, clean format | 2-3 kolom, tabel kompleks |
| Arial/Calibri | Font dekoratif |
| Bullet points sederhana | Ikon, chart, word art |
| Keyword dari JD | Template Canva (sering gagal parsing) |

### Struktur CV
```
HEADER: Nama | Email | No | LinkedIn | Portfolio
RINGKASAN: [posisi] [X thn] [spesialisasi]
PENGALAMAN: [Perusahaan] [Lokasi] [Bulan-Thn] — bullet dengan ANGKA
PENDIDIKAN: Univ | Jurusan | Tahun
SKILL: Technical + Soft Skills
SERTIFIKASI
```

### LinkedIn Optimization
- **Headline**: `[Role] | [Spesialisasi 1] + [Spesialisasi 2] | [Nilai Jual Unik]`
- **About**: Hook 2 kalimat → pengalaman → pencapaian (angka) → tech stack → CTA
- **Experience**: Bullet points + media (screenshot/link)
- **Skills**: 10 paling relevan, minta endorsement
- **Aktivitas**: Post 1-2x/minggu, comment meaningful, follow perusahaan target

### STAR Method
| S — Situation | T — Task | A — Action | R — Result |
|---|---|---|---|
| Konteks situasi | Tugas spesifik | Tindakan kamu | Hasil dengan angka |

### Technical Test Tips
1. Baca soal 2x, tanya clarifying questions
2. Verbalize thinking process
3. Write clean code, test with contoh
4. Optimize setelah working — jangan premature
5. Stuck >15 menit? Skip

### Take-Home Test Timeline
| Day 1 | Day 2-3 | Day 4 | Day 5 | Final |
|-------|---------|-------|-------|-------|
| Baca + plan | Coding inti | Polish + edge cases | Dokumentasi + deploy | Review + submit |

## Career Growth

### Level Progression
| Level | Fokus | Supervisi | Scope |
|-------|-------|-----------|-------|
| Junior (0-2) | Belajar, execute | Dibimbing | Fitur spesifik |
| Mid (2-5) | Independen, solve | Minimal | Modul kompleks |
| Senior (5-8) | Arsitektur, mentor | Zero | System-wide |
| Tech Lead (8+) | Strategi, impact | Mengawasi | Cross-team |

### Salary Benchmark (2024-25)
| Level | Startup | Scale-up | Enterprise |
|-------|---------|----------|------------|
| Junior | Rp 5-8 jt | Rp 7-12 jt | Rp 8-15 jt |
| Mid | Rp 10-18 jt | Rp 15-25 jt | Rp 18-30 jt |
| Senior | Rp 20-35 jt | Rp 28-50 jt | Rp 35-60 jt |
| Tech Lead | Rp 30-50 jt | Rp 45-75 jt | Rp 55-100 jt |

### Negosiasi Gaji
- **Riset**: Glints, Glassdoor, komunitas — tau range pasar
- **Formula**: Minimal (MA) → Ideal (MI) → Impian (MS). Tawaran awal = MI + 15-20%
- **Jangan accept langsung** — minta waktu 1-3 hari
- **Trade-off**: signing bonus, ESOP, remote allowance, training budget

### 5-Year Roadmap
| Tahun | Fokus | Output |
|-------|-------|--------|
| 1 | Foundation | Kuasai 1 bahasa + 1 framework, 3 project |
| 2 | Specialization | Cloud cert, 1 open source PR, tech writing |
| 3-4 | Mastery + Leadership | System design, mentoring, lead project |
| 5 | Impact | Staff/EM/Founder/Consultant — pilih track |

### Hackathon Strategy
| Fase | Waktu | Aktivitas |
|------|-------|-----------|
| Ideas | 3-4 jam | Pahami tema, 10 ide → pilih 1, validasi feasibility |
| Eksekusi | 16-20 jam | MVP dulu, bagi tugas, deploy early, fitur inti selesai 12 jam |
| Final | 4-6 jam | Bug fix critical, polish UI, siapkan demo + pitch 3 menit |

### Open Source — First PR
1. Pilih project yang dipakai sehari-hari
2. Cari issue `good first issue`
3. Comment "I'd like to work on this"
4. Fork → clone → branch → commit (conventional) → push → PR
5. Kontribusi tak harus coding: docs, translation, testing, triage

### Tech Blog
- **Platform**: dev.to, Medium, Hashnode, personal blog
- **Topik laku**: tutorial, case study, comparison, problem solving, career, best practices
- **Konsisten**: 1 artikel per 1-2 minggu
- **Promosi**: share ke LinkedIn, Twitter, komunitas
