**1. Platform freelance apa yang paling cocok untuk developer Indonesia yang ingin memulai project pertama?**

**2. Apa kepanjangan ATS dalam konteks CV lamaran kerja?**

- [ ] Automatic Technical Scanner
- [x] Applicant Tracking System
- [ ] Advanced Talent Sourcing
- [ ] Automated Test Suite

**3. Manakah praktik terbaik untuk optimasi LinkedIn agar dilirik recruiter?**

- [ ] Pakai foto liburan biar terlihat santai
- [ ] Copy paste deskripsi dari CV word ke about section
- [x] Headline jelas (posisi + spesialisasi), about section dengan keyword relevan, dan aktif posting konten
- [ ] Isi skill section dengan semua teknologi yang pernah didengar

**4. Apa metode yang tepat untuk menjawab pertanyaan behavioral interview?**

- [ ] Jelaskan teori sebanyak mungkin
- [x] Gunakan metode STAR (Situation, Task, Action, Result)
- [ ] Ceritakan kehidupan pribadi
- [ ] Berikan jawaban singkat tanpa contoh

**5. Manakah yang BUKAN komponen dalam roadmap karier 5 tahun yang baik?**

- [ ] Target posisi (Junior → Senior → Tech Lead)
- [ ] Target skill yang harus dikuasai setiap tahun
- [ ] Target gaji yang realistis
- [x] Menunggu promosi tanpa belajar hal baru

**6. Apa manfaat utama mengikuti komunitas teknologi seperti IDDev, GDG, atau GitHub Indonesia?**

- [ ] Mendapatkan sertifikat gratis setiap bulan
- [ ] Bisa liburan gratis ke acara luar negeri
- [x] Networking, belajar dari praktisi, kontribusi open source, dan exposure peluang kerja
- [ ] Bisa berhenti belajar mandiri
