# Sesi 1: Freelance Hustle — Mulai Cuan dari Project Pertama

**Durasi:** 90 menit

---

## Tujuan Sesi

Setelah sesi ini, peserta mampu:
- Memilih platform freelance yang sesuai dengan skill dan target pasar
- Membuat profil freelance yang profesional dan meyakinkan
- Menentukan harga (pricing) yang kompetitif dengan rate calculator
- Berkomunikasi dengan klien secara profesional (brief, revisi, payment)
- Memahami aspek pajak dan legal dasar sebagai freelancer

---

## 1. Platform Freelance (15 menit)

### Platform Lokal

| Platform | Keunggulan | Cocok Untuk | Fee Platform |
|----------|-----------|-------------|--------------|
| **Sribu** | Pasar Indonesia, banyak project desain & web | Frontend, UI/UX, content writer, desain grafis | ~20% per project |
| **Fastwork** | Ekosistem Asia Tenggara, sistem escrow | Web dev, mobile, SEO, marketing | 15-20% |
| **Projects.co.id** | Project-based untuk pasar Indonesia | Web dev, aplikasi, IT support | Variatif |
| **Sayaplus** | Komunitas freelancer Indonesia | General freelance | Berdasarkan project |

### Platform Global

| Platform | Keunggulan | Cocok Untuk | Fee Platform |
|----------|-----------|-------------|--------------|
| **Upwork** | Pasar global, payment protection | Developer, designer, writer, virtual assistant | 20% (turunan) |
| **Freelancer** | Contest & project | Developer, data entry | 10-20% |
| **Fiverr** | Gig-based, jasa spesifik | Micro-service, web dev kecil | 20% |
| **Toptal** | High-end, curated talent | Senior dev, arsitek, konsultan | - |

### Tips Memilih Platform

1. **Mulai dari platform lokal** — lebih mudah dapat project pertama
2. **Bangun rating dulu** — 2-3 project kecil untuk portofolio
3. **Expand ke global** — setelah percaya diri dengan bahasa Inggris
4. **Spesifik** — jangan jadi "generalist" di profil, pilih niche

> **Case Study:** Andi, web developer dari Bandung, mulai di Sribu dengan project Rp500K. Setelah 10 project selesai dan rating 5.0, pindah ke Upwork dengan rate $15/jam — dalam 6 bulan sudah $25/jam.

## 2. Profil Freelance (15 menit)

### Komponen Profil Wajib

#### Foto Profil
- Wajah jelas, pencahayaan baik
- Latar belakang netral (putih/abu-abu)
- Senyum, pakaian rapi (bisa kemeja atau casual profesional)
- Hindari: foto selfie di kamar, foto liburan, foto grup

#### Judul / Headline
- Format: `[Posisi] | [Spesialisasi] | [Tahun Pengalaman]`
- Contoh: *Full-Stack Web Developer | React + Laravel | 3 Tahun*
- Contoh: *UI/UX Designer | Figma Expert | 50+ Project Selesai*

#### Deskripsi / Bio
Template bio yang efektif:

```
Halo! Saya [Nama], [Role] dengan [X] tahun pengalaman.

✅ Spesialisasi saya:
- [Skill 1] — [contoh project/sertifikasi]
- [Skill 2] — [contoh project/sertifikasi]
- [Skill 3] — [contoh project/sertifikasi]

✅ Mengapa memilih saya?
- [X] project selesai tepat waktu
- Komunikasi cepat via [platform chat]
- Revisi maksimal [X] kali

✅ Tools yang saya kuasai:
[Tool 1], [Tool 2], [Tool 3], [Tool 4]

Yuk diskusikan project Anda!
```

#### Portfolio
- 3-5 project terbaik
- Sertakan: screenshot, link live, tech stack, durasi pengerjaan
- Jika belum punya project riil: buat project dummy yang menunjukkan skill

#### Testimoni
- Minta testimoni setelah setiap project selesai
- Format: Nama + Jabatan + Perusahaan
- Minta spesifik: *"Cepat, komunikatif, hasil sesuai ekspektasi"*

#### Rate / Tarif
- Cantumkan rate per jam atau per project
- Bisa pakai range: *Rp 100.000 - Rp 250.000 / jam*

## 3. Pricing & Rate Calculator (20 menit)

### Model Pricing

#### 1. Project-Based (Fixed Price)
- Satu harga untuk satu deliverable
- Kelebihan: klien suka pasti, kamu bisa efisien
- Kekurangan: risk scope creep

#### 2. Hourly Rate
- Dibayar per jam kerja
- Kelebihan: adil untuk project yang tidak jelas scope-nya
- Kekurangan: klien khawatir membengkak

#### 3. Retainer
- Bayaran bulanan untuk jumlah jam tetap
- Cocok untuk maintenance, support, content management
- Contoh: 20 jam/bulan × Rp 150.000 = Rp 3.000.000/bulan

### Rate Calculator

Gunakan formula berikut untuk menentukan rate minimum:

```
Biaya Hidup Bulanan = Rp X.XXX.XXX
Biaya Operasional (listrik, internet, tools) = Rp X.XXX.XXX
Tabungan + Investasi (20%) = Rp X.XXX.XXX
Target Pendapatan Bulanan = Total di atas

Jam Kerja Efektif per Bulan = 20 hari × 5 jam = 100 jam

Rate Minimum = Target Pendapatan / 100
Rate Ideal = Rate Minimum × 1.3 (30% margin)
Rate Premium = Rate Minimum × 2 (untuk klien enterprise)
```

**Contoh Perhitungan:**

| Item | Nominal |
|------|---------|
| Biaya hidup | Rp 5.000.000 |
| Operasional | Rp 1.500.000 |
| Tabungan (20%) | Rp 1.300.000 |
| Target bulanan | Rp 7.800.000 |
| Jam efektif | 100 jam |
| **Rate minimum** | **Rp 78.000/jam** |
| **Rate ideal** | **Rp 101.400/jam** |
| **Rate premium** | **Rp 156.000/jam** |

### Benchmark Rate Indonesia (2024-2025)

| Level | Web Developer | Mobile Developer | UI/UX Designer |
|-------|--------------|-----------------|----------------|
| Junior (0-1 thn) | Rp 50-100K/jam | Rp 60-120K/jam | Rp 40-80K/jam |
| Mid (1-3 thn) | Rp 100-200K/jam | Rp 120-250K/jam | Rp 80-150K/jam |
| Senior (3+ thn) | Rp 200-400K/jam | Rp 250-500K/jam | Rp 150-300K/jam |

## 4. Client Communication (20 menit)

### Tahapan Komunikasi

#### 1. Briefing (Awal)
Tanyakan hal-hal ini sebelum memberikan quote:

```
✅ Apa tujuan project ini?
✅ Siapa target audiens?
✅ Ada referensi / contoh yang disukai?
✅ Deadline kapan?
✅ Budget range berapa?
✅ Ada fitur/deliverable yang wajib ada?
✅ Bagaimana format deliverable final?
```

#### 2. Proposal
Template proposal singkat:

```
**Project:** [Nama Project]
**Klien:** [Nama/Klien]
**Scope:**
- [Deliverable 1]
- [Deliverable 2]
- [Deliverable 3]

**Excluded:**
- [Hal yang TIDAK termasuk]

**Timeline:** [X] hari kerja
**Harga:** Rp [X.XXX.XXX]
**Revisi:** [X] kali revisi
**Pembayaran:** 50% DP, 50% setelah selesai
```

#### 3. Selama Pengerjaan
- Update progress setiap 2-3 hari
- Kirim screenshot / preview
- Konfirmasi sebelum lanjut ke tahap berikutnya
- Catat semua perubahan yang diminta

#### 4. Revisi
Aturan revisi yang jelas:

```
✅ "Revisi minor: typo, warna, ukuran — gratis"
✅ "Revisi mayor: perubahan struktur, fitur baru — di luar scope"
✅ "Setelah revisi ke-[X], akan ada charge tambahan Rp [nominal]/revisi"
✅ "Semua revisi harus dalam 1 email / 1 pesan"
```

#### 5. Payment & Escrow
- **Sistem escrow** (Upwork, Fastwork): dana ditahan platform, cair setelah approve
- **Non-escrow** (direct): minta DP 30-50% di awal
- **Payment method:** transfer bank, PayPal, GoPay, Wise
- **Invoice:** kirim invoice resmi sebelum pembayaran

### Contoh Skenario Komunikasi

**Klien:** "Bang, mau minta tambahin fitur login pake Google dong"

**Respon Profesional:**
"Halo [Nama], terima kasih masukannya. Fitur login Google termasuk tambahan di luar scope awal kita. Saya bisa tambahkan dengan biaya Rp XXX.XXX dan estimasi pengerjaan 2 hari. Apakah dilanjutkan? Kalau setuju, saya kirimkan revised quote."

## 5. Tax & Legal untuk Freelancer Indonesia (15 menit)

### NPWP
- Wajib dimiliki freelancer Indonesia
- Cara daftar: Online via ereg.pajak.go.id
- Syarat: KTP, KK, akta kelahiran
- Gratis, proses 1-3 hari kerja

### Jenis Pajak

#### PPh Final (PP 23/2018) — 0.5%
- Untuk omzet di bawah Rp 4.8 miliar/tahun
- Paling umum untuk freelancer kecil
- Bayar via e-Billing atau e-Filing

#### PPh 21 — Tarif progresif
- Untuk pegawai tetap (bukan freelancer murni)

#### PPh 23 — 2%
- Dipotong oleh klien badan usaha
- Contoh: klien PT memotong 2% dari nilai invoice
- Kamu terima bersih 98%, dapat bukti potong

### Invoice

Komponen invoice yang benar:

```
**[NAMA KAMU / NAMA USAHA]**
[Nomor Telepon] | [Email]
[Alamat]
NPWP: [XX.XXX.XXX.X-XXX.XXX]

**INVOICE No: INV/2025/001**

Kepada:
[Nama Klien / Perusahaan]
[Alamat Klien]

**Detail:**
1. [Deskripsi pekerjaan] — Rp X.XXX.XXX
   Total: Rp X.XXX.XXX

**Pembayaran:**
Bank: [Nama Bank]
Rekening: [No Rek] a.n. [Nama]
Total: Rp X.XXX.XXX

Terbilang: [...]

Hormat saya,
[Nama Kamu]
```

### Tips Legal
- Simpan semua kontrak/SPK (Surat Perintah Kerja)
- Screenshot percakapan penting
- Gunakan kontrak sederhana untuk project > Rp 5 juta
- Pertimbangkan buat PT Perorangan (UU Cipta Kerja) jika omzet > Rp 50 juta/tahun

## 6. Latihan Praktik (5 menit)

### Latihan 1: Bikin Profil Freelance
1. Pilih salah satu platform (Sribu/Fastwork/Upwork)
2. Buat draft profil lengkap dengan:
   - Foto profil
   - Headline
   - Bio (gunakan template)
   - Portfolio (minimal 3 project atau dummy project)
3. Kirim ke mentor untuk review

### Latihan 2: Hitung Rate Kamu
1. Hitung biaya hidup bulanan
2. Hitung jam kerja efektif
3. Gunakan rate calculator untuk dapat:
   - Rate minimum
   - Rate ideal
   - Rate premium
4. Bandingkan dengan benchmark rate Indonesia
5. Tentukan rate yang kamu tawarkan

### Latihan 3: Roleplay Client Communication
Bersama partner:
1. Satu orang jadi klien dengan brief project website UMKM
2. Satu orang jadi freelancer yang harus:
   - Menggali requirements
   - Membuat proposal
   - Menangani permintaan revisi di luar scope
3. Tukar peran
