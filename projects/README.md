## 🎯 Tujuan Showcase

Folder ini menyajikan **contoh hasil jadi** dari proyek capstone yang dikerjakan mahasiswa. Setiap file showcase mendokumentasikan:

1. **Arsitektur sistem** — diagram alur frontend → backend → AI agent → database
2. **Tech stack lengkap** — dengan versi dan konfigurasi
3. **Fitur unggulan** — screenshot placeholder + deskripsi fungsional
4. **Code snippet penting** — cuplikan kode production-ready (agent definition, tools, pipeline)
5. **Link demo & repo** — placeholder yang bisa diganti dengan URL hasil deployment

> 💡 **Catatan:** Showcase ini bersifat **referensi dan inspirasi**. Mahasiswa didorong untuk mengembangkan project dengan fitur tambahan, UI yang lebih baik, dan inovasi AI yang lebih advanced.

## 🛠️ Stack Umum

Semua capstone menggunakan stack inti yang sama:

```
Frontend    →  React / Next.js + Tailwind CSS
Backend     →  Express / Hono + TypeScript
AI Agent    →  Mastra AI Framework
LLM         →  OpenAI GPT-4o-mini / Gemini
Database    →  PostgreSQL + Prisma / Drizzle ORM
Deploy      →  Vercel (FE) + Railway / Render (BE)
```

## 🚀 Cara Menggunakan Showcase Ini

**Sebagai referensi belajar:**
- Baca file showcase project yang sesuai minatmu
- Pelajari arsitektur dan alur data
- Jalankan code snippet di local environment

**Sebagai template presentasi:**
- Copy file `.md` yang relevan
- Ganti screenshot placeholder dengan screenshot asli projectmu
- Update link demo dan GitHub repo

**Sebagai portofolio:**
- Gunakan sebagai draft dokumentasi project
- Tambahkan hasil evaluasi dan metrik performa
- Sertakan link video demo
