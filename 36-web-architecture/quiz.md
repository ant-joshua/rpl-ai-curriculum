# Quiz: Web Architecture

<div class="quiz">

**1. HTTP method `GET` seharusnya bersifat:**

- [x] Idempotent dan safe — tidak mengubah data di server
- [ ] Menghapus resource
- [ ] Mengirim data ke server untuk diproses
- [ ] Memperbarui resource secara parsial

**2. HTTP status code 404 berarti:**

- [ ] Internal Server Error
- [x] Not Found — resource yang diminta tidak ada
- [ ] Unauthorized
- [ ] Created

**3. DNS (Domain Name System) berfungsi untuk:**

- [ ] Menyimpan file website
- [x] Menerjemahkan domain (contoh.com) ke IP address server
- [ ] Mengenkripsi koneksi browser
- [ ] Mengatur load balancing

**4. Perbedaan utama frontend dan backend adalah:**

- [x] Frontend berjalan di browser (UI/UX), backend berjalan di server (logika bisnis, database, API)
- [ ] Frontend lebih sulit dari backend
- [ ] Backend hanya berisi database
- [ ] Frontend tidak membutuhkan internet

**5. Shared hosting berbeda dengan VPS karena:**

- [x] Shared hosting: satu server dipakai banyak pengguna, resource terbatas. VPS: partisi virtual dengan resource dedicated
- [ ] VPS lebih murah dari shared hosting
- [ ] Shared hosting memberikan akses root penuh
- [ ] VPS tidak bisa install custom software

**6. Manakah urutan yang benar saat browser memuat halaman web?**

- [ ] Render page → Parse HTML → Download CSS/JS → DNS lookup
- [x] DNS lookup → TCP handshake → HTTP request → Parse HTML → Load resources → Render
- [ ] Download CSS → Parse HTML → DNS lookup → Render
- [ ] HTTP request → DNS lookup → Render → Parse HTML
