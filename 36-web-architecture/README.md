> **Level:** 🌱 Beginner  
> **Jam:** 6 (4 sesi)  
> **Prasyarat:** — (lebih baik setelah 00-fundamentals)

## Tujuan Pembelajaran

Setelah modul ini, kamu bisa:

- Paham cara kerja web: client, server, request-response cycle
- Bedain frontend vs backend dan tugas masing-masing
- Ngerti HTTP methods, status codes, dan headers
- Paham API dan REST, bisa tes endpoint pake Postman
- Tahu cara hosting static site dan bedain jenis hosting
- Baca URL, pake browser devtools, analisis network tab
