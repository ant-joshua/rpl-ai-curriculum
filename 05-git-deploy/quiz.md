# Quiz: Git & Deployment

> Jawab pertanyaan di bawah dengan klik pilihan yang benar.

<div class="quiz">

**1. Perintah untuk menginisialisasi repository Git baru?**

- [ ] `git start`
- [x] `git init`
- [ ] `git new`
- [ ] `git create`

**2. Setelah `git add .`, perintah untuk menyimpan perubahan ke history?**

- [ ] `git push`
- [x] `git commit -m "pesan"`
- [ ] `git save`
- [ ] `git store`

**3. Perintah untuk membuat branch baru bernama `fitur-a`?**

- [x] `git branch fitur-a`
- [ ] `git new-branch fitur-a`
- [ ] `git checkout fitur-a`
- [ ] `git branch -d fitur-a`

**4. Setelah selesai di branch `fitur-a`, bagaimana menggabungkan ke `main`?**

- [ ] `git merge main` (saat di fitur-a)
- [x] `git checkout main && git merge fitur-a`
- [ ] `git combine fitur-a main`
- [ ] `git push fitur-a main`

**5. Perintah untuk mengirim commit lokal ke GitHub?**

- [x] `git push`
- [ ] `git pull`
- [ ] `git upload`
- [ ] `git send`

**6. `git pull` melakukan...**

- [x] `git fetch` dilanjut `git merge`
- [ ] `git push` kebalikan
- [ ] Menghapus branch remote
- [ ] Membuat branch baru

**7. Vercel adalah platform untuk...**

- [ ] Menulis kode
- [x] Deploy frontend dan serverless functions
- [ ] Version control
- [ ] Database hosting

**8. Cara kolaborasi di GitHub tanpa akses langsung ke repo utama?**

- [ ] Langsung push ke main
- [x] Fork repo → clone → buat perubahan → buat Pull Request
- [ ] Email kode ke owner
- [ ] Copy paste file

**9. Conflict di Git terjadi ketika...**

- [ ] Dua orang commit di waktu sama
- [x] Dua branch mengubah baris yang sama pada file yang sama
- [ ] File terlalu besar
- [ ] Koneksi internet putus

**10. Perintah untuk melihat status file yang dimodifikasi tapi belum di-stage?**

- [ ] `git log`
- [x] `git status`
- [ ] `git diff`
- [ ] `git show`

</div>
