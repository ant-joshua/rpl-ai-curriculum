## Pertemuan 1: GitHub Profesional — Profile README, Pinned Repos, Badges Shields.io, Dokumentasi Proyek, GitHub Pages

### Tujuan
- Membuat GitHub Profile README profesional dengan badge & statistik
- Mengelola pinned repositories & dokumentasi proyek
- Setup GitHub Pages untuk project site
- Menggunakan Shields.io badges untuk skill & tools

### Kegiatan (90 menit)
| Waktu | Aktivitas | Metode | Media |
|-------|-----------|--------|-------|
| 10' | Apersepsi & review: tanya pengalaman GitHub, demo profile developer terkenal sebagai inspirasi | Tanya jawab | Slide, browser (GitHub) |
| 20' | Materi inti: GitHub Profile README (special repo), Markdown styling, badge Shields.io, GitHub stats widget, pinned repos, README dokumentasi proyek, GitHub Pages setup | Ceramah + demo | Live code, GitHub |
| 25' | Praktik terbimbing: buat repo profile username + Profile README dengan badge skill + statistik + pinned project | Hands-on | Starter template |
| 20' | Latihan mandiri: tambah GitHub Pages site + dokumentasi proyek lengkap (README, kontribusi, lisensi) | Problem solving | Soal |
| 15' | Diskusi & refleksi: review profile peserta, tips dokumentasi proyek agar standout, GitHub sebagai portfolio | Q&A | — |
