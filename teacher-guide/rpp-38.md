## Pertemuan 1: AI Coding Tools — Landscape, Setup, Code Generation, AI Refactor

### Tujuan
- Memahami lanskap alat bantu coding berbasis AI (Cline, Copilot, Cursor, Claude Code)
- Setup & konfigurasi AI coding tools di local environment
- Menggunakan AI untuk code generation & refactoring

### Kegiatan (90 menit)
| Waktu | Aktivitas | Metode | Media |
|-------|-----------|--------|-------|
| 10' | Apersepsi & review: demo AI coding tools, tanya pengalaman siswa pakai AI | Tanya jawab | Slide, browser |
| 20' | Materi inti: landscape AI tools (Cline, Copilot, Cursor, Claude Code), perbandingan fitur | Ceramah + demo | Slide, live demo |
| 25' | Praktik terbimbing: setup Cline/Copilot di VS Code + generate kode pertama dari prompt | Hands-on | VS Code, terminal |
| 20' | Latihan mandiri: prompt code generation, prompt refactor, prompt optimasi | Problem solving | AI tool |
| 15' | Diskusi & refleksi: mana tool paling cocok buat use case berbeda | Q&A | — |
