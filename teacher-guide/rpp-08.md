### Tujuan
- Merencanakan proyek full-stack dengan GitHub Project Board
- Membuat ERD (Entity Relationship Diagram)
- Setup project structure

### Kegiatan (90 menit)

| Waktu | Aktivitas | Metode | Media |
|-------|-----------|--------|-------|
| 10' | **Apersepsi**: "Aplikasi apa yang mau kalian bangun?" — brainstorm topik | Brainstorming | Whiteboard |
| 20' | **Planning**: GitHub Project Board, user stories, ERD | Ceramah + praktik | GitHub, dbdiagram.io |
| 15' | **Tech stack decision**: Express + React/Next.js + Mastra + PostgreSQL | Diskusi kelompok | Slide opsi stack |
| 20' | **Setup project**: init repos, tsconfig, folder structure, DB migration | Praktik | Terminal |
| 20' | **Praktik**: Setup backend + frontend skeleton | Hands-on | Starter template |
| 5' | **Review sprint plan**: Apa yang harus selesai minggu depan? | Q&A | GitHub Board |
