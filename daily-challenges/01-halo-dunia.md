# 01. Halo Dunia 🌱

**Level:** Mudah  
**Topik:** Variabel & Output

## 🎯 Learning Objective
Mengenal `console.log()` dan deklarasi variabel dasar.

## 📝 Problem
Buat program yang mencetak "Halo, Dunia!" ke konsol, lalu cetak juga nama kamu.

## 🧪 Starter Code

```javascript
// Tulis kode kamu di sini
let nama = "...";
console.log("Halo, Dunia!");
console.log("Nama saya " + ...);
```

## ✅ Expected Output

```
Halo, Dunia!
Nama saya Budi
```

## 💡 Hint
Gunakan operator `+` untuk menggabungkan teks (string concatenation).

<details>
<summary>Klik untuk lihat solusi</summary>

```javascript
let nama = "Budi";
console.log("Halo, Dunia!");
console.log("Nama saya " + nama);

// Alternatif dengan template literal:
console.log(`Nama saya ${nama}`);
```
</details>
