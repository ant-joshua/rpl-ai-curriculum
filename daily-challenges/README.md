# 🧩 Daily Coding Challenges

Selamat datang di **Daily Coding Challenges** — latihan harian untuk menguasai pemrograman JavaScript dari dasar hingga mahir.

## Cara Penggunaan

1. Kerjakan 1 challenge per hari (atau sesuai ritme kamu)
2. Baca soal, pahami learning objective
3. Tulis kode di file `.js` terpisah
4. Jalankan dengan `node namafile.js` atau buka di browser untuk soal DOM
5. Cek solusi hanya **setelah** kamu mencoba sendiri
6. Catat progres di jurnal belajar

## Tingkat Kesulitan

| Level | Bahasa | Rentang | Durasi |
|-------|--------|---------|--------|
| 🌱 **Mudah** | Easy | 01 — 15 | 10–20 menit |
| 🔥 **Sedang** | Medium | 16 — 25 | 20–40 menit |
| 💀 **Sulit** | Hard | 26 — 30 | 30–60 menit |

## Topik yang Dicakup

Variabel, Tipe Data, Conditionals, Loops, Arrays, Objects, Functions, String Manipulation, DOM Basics, Array Methods, Async Basics, Error Handling, Regex.
