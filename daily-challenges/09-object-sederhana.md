# 09. Object Sederhana 🌱

**Level:** Mudah  
**Topik:** Object

## 🎯 Learning Objective
Membuat dan mengakses properti object.

## 📝 Problem
Buat object `siswa` dengan properti: nama, umur, kelas. Cetak setiap properti ke konsol.

## 🧪 Starter Code

```javascript
let siswa = {
  nama: "...",
  umur: ...,
  kelas: "..."
};

console.log("Nama:", ...);
console.log("Umur:", ...);
console.log("Kelas:", ...);
```

## ✅ Expected Output

```
Nama: Budi
Umur: 17
Kelas: XI RPL 1
```

## 💡 Hint
Akses properti dengan dot notation: `object.properti`.

<details>
<summary>Klik untuk lihat solusi</summary>

```javascript
let siswa = {
  nama: "Budi",
  umur: 17,
  kelas: "XI RPL 1"
};

console.log("Nama:", siswa.nama);
console.log("Umur:", siswa.umur);
console.log("Kelas:", siswa.kelas);
```
</details>
