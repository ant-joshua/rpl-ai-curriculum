# Deploy Frontend ke Vercel

1. Push project ke GitHub
2. Buka vercel.com
3. Login pake GitHub
4. Klik "Add New Project"
5. Pilih repo
6. Settings: Framework = Vite/React, Build = npm run build, Output = dist
7. Deploy -> `namaproject.vercel.app`

**Auto-deploy:** Tiap push ke main, Vercel auto deploy.
