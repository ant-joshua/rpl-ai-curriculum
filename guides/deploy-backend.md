# Deploy Backend ke Railway

1. Push project ke GitHub
2. Buka railway.app
3. Login pake GitHub
4. "New Project" > "Deploy from GitHub"
5. Pilih repo
6. Set Environment Variables (DATABASE_URL, OPENAI_API_KEY, dll)
7. Deploy -> `namaproject.up.railway.app`

**PostgreSQL:** Railway kasih PostgreSQL gratis.
