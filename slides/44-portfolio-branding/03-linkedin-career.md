# Sesi 3: LinkedIn & Career

> Optimasi profil LinkedIn dan CV untuk menarik perekrut, plus bangun personal branding statement yang kuat.

**Durasi**: 4 jam | **Output**: Profil LinkedIn optimal + CV siap pakai

---

## 3.1 LinkedIn Profile Optimization

Profil LinkedIn yang optimal meningkatkan visibilitas di pencarian perekrut hingga 40x.

### Headline

Headline muncul di bawah nama — ini elemen pertama dilihat perekrut.

**Format optimal:**
```
[Jabatan] | [Spesialisasi] | [Skill Utama] | [Value Proposition]
```

**Contoh:**

| ❌ Kurang Baik | ✅ Optimal |
|----------------|-----------|
| Frontend Developer at PT Maju | Frontend Developer | React & TypeScript | Building Performant Web Apps |
| Mahasiswa | Computer Science Student | Full-Stack Dev | React | Python | Building for Impact |
| Looking for job | Frontend Developer | 3+ Years Exp | React Specialist | Open to Work |

### About Section (2-3 paragraf)

**Template Paragraf 1 — Hook:**
> "Frontend Developer with [angka] years of experience building [jenis aplikasi] using [teknologi utama]. Passionate about creating [value] through [pendekatan teknis]."

**Template Paragraf 2 — Bukti:**
> "At [perusahaan/proyek], I [action verb] [what you did], resulting in [angka hasil]. Previously at [previous role/project], I [achievement lain]."

**Template Paragraf 3 — CTA:**
> "Currently exploring opportunities in [bidang/industri]. Open to [jenis kolaborasi]. Let's connect if you're interested in [topik]!"

### Featured Section

Gunakan fitur **Featured** untuk menampilkan:

1. **Portfolio website** — tambahkan link ke live site
2. **GitHub profile** — tambahkan link GitHub
3. **Artikel atau project** — tambahkan post/case study
4. **Sertifikasi** — screenshots sertifikat

Cara: Buka profil → Klik **Add a featured section** → Pilih Media / Links / Posts

### Skills & Endorsement

Strategi skills endorsement:

- **Pilih 5-10 skill utama** yang relevan dengan target karir
- Susun urutan skill — skill paling kuat di atas
- **Minta endorsement** dari teman, rekan kerja, dosen
- Ambil **LinkedIn Skill Assessment** untuk dapat badge "Skill Verified"

**Skill Wajib Developer:**
| Kategori | Skill |
|----------|-------|
| Bahasa | JavaScript, TypeScript, Python, Go |
| Frontend | React, Next.js, Vue, Tailwind CSS |
| Backend | Node.js, Express, Django, PostgreSQL |
| Tools | Git, Docker, CI/CD, AWS, Figma |
| Soft Skills | Problem Solving, Team Collaboration, Communication |

### Experience Section

Format bullet untuk job/project experience:

```
[Action Verb] + [What you did] + [How you did it] + [Result/Impact]
```

**Contoh:**
- Developed responsive web dashboard using React and TypeScript, reducing page load time by 35%
- Led migration from REST to GraphQL API, improving data fetching efficiency by 50% across 10+ endpoints
- Collaborated with 5 cross-functional team members to ship 3 major features in Q1 2026

### Action Verbs (dari ATS-friendly)

| Kategori | Verbs |
|----------|-------|
| Leadership | Led, Directed, Spearheaded, Coordinated, Managed |
| Development | Built, Developed, Engineered, Architected, Implemented |
| Analysis | Analyzed, Evaluated, Assessed, Optimized, Audited |
| Design | Designed, Created, Architected, Structured, Formulated |
| Improvement | Improved, Enhanced, Streamlined, Transformed, Refactored |

### Connection Strategy

- **Target 500+ connections** — threshold LinkedIn "All-Star" profile
- Kirim **personalized connection request**
- Contoh: *"Hi [name], Saya melihat background kamu di [bidang]. Saya juga tertarik di [topik]. Would love to connect and follow your journey!"*
- **Engage** — like, comment, share post minimal 3x seminggu

## 3.2 ATS-Friendly CV / Resume

ATS (Applicant Tracking System) adalah software yang membaca CV secara otomatis. 75% CV ditolak ATS sebelum dilihat manusia.

### Format ATS Checklist

| Aturan | ✅ Lakukan | ❌ Jangan |
|--------|-----------|-----------|
| Font | Arial, Calibri, Roboto (10-12pt) | Script, decorative, gambar font |
| File | **PDF** (save as standard PDF) | JPG, PNG, Canva export (images) |
| Structure | Single column, clear headings | Multi-column, tables, text boxes |
| Section | Standard: Summary → Experience → Education → Skills | Creative layouts, icons as bullets |
| Keywords | Match job description keywords | Generic skills without context |
| Header | Nama + phone + email + LinkedIn + GitHub | Foto, alamat lengkap, tanggal lahir |

### Template CV ATS-Friendly

```
[NAMA LENGKAP]
[Phone] | [Email] | linkedin.com/in/[username] | github.com/[username]
[Portfolio URL]

---
PROFESSIONAL SUMMARY
[2-3 kalimat: role + experience + key skills + target]

EXPERIENCE
[Job Title] | [Company] | [Location]
[Month Year] – [Month Year]

- [Action verb] [task] using [tools], resulting in [metric]
- [Action verb] [task] using [tools], resulting in [metric]
- [Action verb] [task] using [tools], resulting in [metric]

[Previous Job Title] | [Company] | [Location]
[Month Year] – [Month Year]

- [Same format — action, tool, result]
- ...

EDUCATION
[Degree] in [Major] | [University]
[Year] – [Year]
- GPA: [value] (optional)
- Relevant coursework: [list]

PROJECTS
[Project Name] — [Tech Stack]
- Brief description + impact
- Link: github.com/username/project

SKILLS
Languages: JavaScript, TypeScript, Python
Frameworks: React, Next.js, Node.js, Express
Tools: Git, Docker, AWS, PostgreSQL, Figma

CERTIFICATIONS
- [Cert Name] | [Issuer] | [Year]
- [Cert Name] | [Issuer] | [Year]

---
```

### Achievement with Numbers

Ganti deskripsi tugas biasa dengan achievement yang terukur:

| ❌ Tugas Biasa | ✅ Achievement Terukur |
|----------------|----------------------|
| Developed website using React | Built React dashboard with 15+ components, serving 1,000+ daily users |
| Fixed bugs in codebase | Identified and resolved 40+ production bugs, improving app stability by 30% |
| Worked on team projects | Collaborated in 5-person agile team to deliver 3 sprints on time |
| Wrote unit tests | Achieved 85% code coverage with 200+ unit tests using Jest |
| Improved performance | Optimized API queries reducing response time from 2.3s to 0.4s (82% faster) |

### One-Page Rule

CV maksimal **1 halaman** untuk < 10 tahun pengalaman. Cara ringkas:

- Hanya 5-7 bullet terbaik per role
- Hilangkan deskripsi tugas yang obvious
- Gabungkan proyek kecil jadi satu section
- Gunakan angka untuk highlight impact

## 3.3 Personal Branding Statement

Personal branding statement adalah satu kalimat yang menjelaskan: siapa kamu, apa yang kamu lakukan, dan kenapa itu penting.

### Formula

```
I help [target audience] [achieve outcome] through [your expertise/method].
```

**Contoh:**
- "I help startups build fast, accessible web applications using React and TypeScript."
- "I help businesses turn complex data into actionable insights with clean dashboards."
- "I help junior developers transition into tech through structured mentoring and real-world projects."

### Untuk LinkedIn Headline

```
[Role] | [Skill 1] + [Skill 2] | [Impact Statement]
```

**Contoh:**
- "Frontend Developer | React + TypeScript | Building performant web apps"
- "Software Engineer | Node.js + PostgreSQL | Scaling backend systems"
- "UI/UX Developer | Figma → React | Designing & building user-centric interfaces"

### Untuk CV Summary

```
[Role] with [X years] experience in [field]. Specialized in [skill 1], [skill 2], and [skill 3]. 
Proven track record of [achievement type], including [specific result]. 
Seeking to leverage [strength] at [target company type].
```

**Contoh:**
> "Frontend Developer with 3+ years experience in building web applications. Specialized in React, TypeScript, and Tailwind CSS. Proven track record of improving user experience and performance, including reducing page load time by 40% and increasing user engagement by 25%. Seeking to leverage modern frontend expertise at a fast-growing SaaS company."

## 3.4 LinkedIn Profile Checklist

Gunakan checklist sebelum dianggap selesai:

- [ ] **Profile photo** profesional (bukan selfie, bukan liburan)
- [ ] **Background banner** relevan (tech theme, personal brand)
- [ ] **Headline** mengandung keyword pencarian role target
- [ ] **About section** 3 paragraf (hook, bukti, CTA)
- [ ] **Featured** — minimal 2 item (portfolio, GitHub, artikel)
- [ ] **Experience** — semua bullet pakai action verb + angka
- [ ] **Education** — diisi lengkap
- [ ] **Skills** — minimal 5 skill, urut sesuai prioritas
- [ ] **Licenses & Certifications** — diisi jika ada
- [ ] **URL** — custom LinkedIn URL (`linkedin.com/in/namakamu`)
- [ ] **Open to Work** — aktif (bisa visible ke semua atau hanya recruiters)

## ✍️ Latihan

### Latihan 1: Optimasi Headline & About

Buka profil LinkedIn kamu (atau buat draft).

1. Tulis **headline** baru menggunakan format: `[Role] | [Skill 1] + [Skill 2] | [Value Proposition]`
2. Tulis **About section** 3 paragraf:
   - Paragraf 1: Hook — siapa kamu dan apa yang kamu lakukan
   - Paragraf 2: Bukti — pencapaian dengan angka
   - Paragraf 3: CTA — apa yang kamu cari
3. Kirim ke teman/dosen untuk feedback

**Kriteria**: Headline mengandung keyword yang bisa dicari perekrut, about section maksimal 300 kata.

### Latihan 2: Rewrite Experience Bullets

Ambil 3 pengalaman (kerja/ magang/ organisasi/ proyek kampus) kamu.

Untuk setiap pengalaman, tulis ulang bullet points menggunakan format:
```
[Action Verb] + [What] + [Tools] + [Result with numbers]
```

**Contoh transformasi:**
| Sebelum | Sesudah |
|---------|---------|
| "Membuat website company profile" | "Built company profile website using React and Tailwind CSS, resulting in 40% increase in online inquiries" |
| "Mengelola database" | "Managed PostgreSQL database with 10+ tables, serving 500+ daily transactions with 99.9% uptime" |

**Kriteria**: Minimal 9 bullet (3 per role), setiap bullet ada angka/impact.

### Latihan 3: Buat CV ATS-Friendly

Gunakan template ATS-friendly dari section 3.2. Isi dengan data diri:

1. Professional Summary (2-3 kalimat)
2. Experience (minimal 1 role, 3 bullets per role)
3. Education
4. Projects (minimal 2)
5. Skills (kategorikan: Languages, Frameworks, Tools)
6. Certifications (jika ada)

Export sebagai PDF. Cek menggunakan [resumeworded.com](https://resumeworded.com) atau tools serupa.

**Kriteria**: Single column, no images/icons, 1 page, PDF format, ATS score minimal 70.

### Latihan 4: Personal Branding Statement + All-Star Profile

1. Tulis **personal branding statement** menggunakan formula:
   `I help [target] [outcome] through [method].`
2. Tulis ulang **LinkedIn headline** berdasarkan statement tersebut
3. Apply **LinkedIn profile checklist** (section 3.4) — pastikan semua centang terpenuhi
4. Kirim **personalized connection request** ke 5 orang di bidang yang sama

**Kriteria**: Personal branding statement jelas dan spesifik (bukan generik), LinkedIn profile minimal level "Intermediate" atau "All-Star", connection request tidak copy-paste massal.
