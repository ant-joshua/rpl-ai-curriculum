# Sesi 3: Career Growth — Dari Junior ke Tech Leader

**Durasi:** 90 menit

---

## Tujuan Sesi

Setelah sesi ini, peserta mampu:
- Memahami perbedaan skill dan tanggung jawab di setiap level karier
- Membuat skill roadmap realistis untuk 5 tahun ke depan
- Mengetahui benchmark gaji industri teknologi di Indonesia
- Melakukan negosiasi gaji dengan percaya diri
- Mengembangkan soft skills esensial untuk jenjang karier

---

## 1. Junior → Senior → Lead (15 menit)

### Perbandingan Level

| Aspek | Junior (0-2 thn) | Mid (2-5 thn) | Senior (5-8 thn) | Lead / Tech Lead (8+ thn) |
|-------|-----------------|---------------|-----------------|--------------------------|
| **Fokus** | Belajar, execute task | Independent, solve problem | Arsitektur, mentor | Strategi, tim, impact |
| **Supervisi** | Dibimbing senior | Minimal supervisi | Zero supervisi | Mengawasi orang lain |
| **Scope** | Fitur spesifik | Modul / fitur kompleks | System / service wide | Cross-team / org |
| **Code review** | Di-review | Mereview junior | Mereview semua | Arch design review |
| **Decision** | Teknis kecil | Teknis medium | Teknis besar + trade-off | Teknis + business |
| **Komunikasi** | Tim internal | Tim + lead | Tim + stakeholder | Multi-tim + leadership |
| **Kesalahan** | Dimaklumi, learning | Tanggung jawab sendiri | Impact besar | Blame culture jika gagal |

### Skill per Level

#### Junior — Foundation
```
Technical:
✅ Satu bahasa pemrograman dikuasai
✅ Basic CRUD, API consumption
✅ Version control (Git)
✅ Satu framework (React/Laravel/Django)
✅ SQL dasar

Non-technical:
✅ Follow instructions dengan tepat
✅ Rajin bertanya jika bingung
✅ Dokumentasi, commit message
✅ Time management, deadline
```

#### Mid-Level — Independence
```
Technical (tambah):
✅ Design patterns (MVC, Singleton, Factory)
✅ Testing (unit, integration)
✅ Database design & optimization
✅ REST API design
✅ Docker, CI/CD dasar
✅ Satu cloud platform (AWS/GCP/Azure)

Non-technical:
✅ Code review untuk junior
✅ Estimasi task akurat
✅ Komunikasi dengan non-teknis
✅ Problem solving mandiri
```

#### Senior — Impact
```
Technical (tambah):
✅ System design & architecture
✅ Microservices / monolith trade-off
✅ Performance optimization
✅ Security best practices
✅ Multiple tech stacks
✅ Production incident handling

Non-technical:
✅ Mentoring junior & mid
✅ Tech decision making
✅ Stakeholder management
✅ Project estimation & planning
✅ Code review standards
```

#### Tech Lead — Strategy
```
Technical:
✅ Architecture decision records (ADR)
✅ Platform / infrastructure strategy
✅ Technical debt management
✅ Scalability & reliability patterns

Non-technical:
✅ Team leadership & coaching
✅ Cross-team collaboration
✅ Hiring & interview panels
✅ Engineering culture building
✅ Budget & resource planning
✅ Conflict resolution
```

## 2. Skill Roadmap — 5 Tahun (20 menit)

### Tahun 1: Foundation & Exploration

| Quarter | Fokus | Output |
|---------|-------|--------|
| Q1 | Kuasai 1 bahasa + 1 framework | Mini project (CRUD app) |
| Q2 | Version control, deployment dasar | Deploy app ke VPS/Cloud |
| Q3 | Database, API, testing | Fitur kompleks dengan testing |
| Q4 | Portfolio, freelance, job prep | 3 project di portfolio + CV siap |

**Target:**
- Selesai 2-3 kursus/sertifikasi
- 3 project di GitHub
- Ikut 1-2 hackathon / coding challenge
- Apply 20+ lowongan

### Tahun 2: Independence & Specialization

| Quarter | Fokus | Output |
|---------|-------|--------|
| Q1 | Dalami 1 spesialisasi (FE/BE/DEVOPS) | Sertifikasi spesialisasi |
| Q2 | Cloud fundamentals (AWS/GCP) | 1 cloud certification |
| Q3 | Open source contribution | 1 merged PR ke project populer |
| Q4 | Public speaking / tech writing | 1 artikel atau 1 tech talk |

**Target:**
- Bisa handle project sendiri (end-to-end)
- Mulai mentoring 1 junior
- Naik gaji 20-30%
- 1+ kontribusi open source

### Tahun 3-4: Mastery & Leadership

| Quarter | Fokus | Output |
|---------|-------|--------|
| Q1 | System design & architecture | Sertifikasi system design |
| Q2 | Leadership fundamentals | Mentoring 2-3 junior/mid |
| Q3 | Cross-functional collaboration | Lead 1 cross-team project |
| Q4 | Career pivot decision | Senior / Staff engineer role |

**Target:**
- Bisa menjadi go-to person untuk area spesifik
- Memimpin 1 project besar
- Networking aktif (komunitas, conference)
- Naik gaji 50% dari tahun 1

### Tahun 5: Impact & Influence

- **Track A — Technical:** Staff / Principal Engineer
- **Track B — Management:** Engineering Manager / Tech Lead Manager
- **Track C — Entrepreneurship:** Startup founder / CTO early-stage
- **Track D — Freelance/Consultant:** Rate $50-100/jam

## 4. Negosiasi Gaji (15 menit)

### Kenapa Negosiasi Penting?

- **1% kenaikan di awal** = puluhan juta selisih dalam 5 tahun
- Perempuan dan minoritas sering tidak negotiate — rugi besar
- Perusahaan *expect* kamu negotiate — jangan malu

### Research Rate Sebelum Negosiasi

```
Sebelum negosiasi, kamu harus tahu:

✅ Rata-rata gaji untuk posisi (dari Glints, Glassdoor, komunitas)
✅ Budget perusahaan untuk posisi tersebut (tanya di awal)
✅ Nilai pasar kamu (berdasarkan skill dan pengalaman)
✅ BATNA — Best Alternative to Negotiated Agreement
```

### Formula Target Gaji

```
Gaji Minimal (MA): Rp X — tidak mau di bawah ini
Gaji Ideal (MI): Rp X — senang terima
Gaji Impian (MS): Rp X — kalau dapat, amazing

Tawaran awal = MI + 15-20% (ada ruang negosiasi)
```

### Timing Negosiasi

| Fase | Aktivitas |
|------|-----------|
| **Awal proses** | Tanya range budget: *"Sebelum lanjut, bisa share budget range untuk posisi ini?"* |
| **Sebelum tawaran** | Tunjukkan value, bukan butuh |
| **Saat tawaran datang** | Jangan accept langsung — minta waktu 1-3 hari |
| **Counter offer** | Berikan angka dengan justifikasi |
| **Final** | Jika deal, minta kontrak tertulis |

### Script Negosiasi

**Saat dapat tawaran:**

> *"Terima kasih atas tawarannya! Saya sangat antusias dengan kesempatan ini. Sebelum saya terima, saya ingin diskusi tentang kompensasi. Berdasarkan riset saya dan pengalaman yang saya miliki, saya berharap ada di kisaran [X juta] - [Y juta]. Apakah ada fleksibilitas untuk angka tersebut?"*

**Jika budget tidak bisa naik:**

> *"Saya paham. Mungkin kita bisa diskusi komponen lain? Seperti signing bonus, opsi saham, atau benefit training?"*

### Trade-offs Selain Gaji

| Benefit | Nilai |
|---------|-------|
| Signing bonus | 1x gaji sekali di awal |
| Stock options / ESOP | Potensi besar jika IPO |
| Remote / WFH allowance | Hemat transport + sewa |
| Training & certification budget | Rp 5-20 juta/tahun |
| Internet & device allowance | Rp 500K - 2 juta/bulan |
| Flexible hours | Quality of life |
| Additional leave | Work-life balance |
| Performance bonus | 1-3 bulan gaji/tahun |

## 5. Soft Skills untuk Karier (15 menit)

### Komunikasi

| Skill | Deskripsi | Cara Latihan |
|-------|-----------|--------------|
| **Teknis → Non-teknis** | Jelaskan konsep tanpa jargon | Coba jelaskan coding ke teman non-teknis |
| **Tulisan** | Dokumentasi, email, proposal | Blogging, README, meeting notes |
| **Presentasi** | Tech talk, demo, standup | Latihan presentasi 5 menit tiap minggu |
| **Asynchronous** | Komunikasi via chat, email, PR | Tulis pesan lengkap dalam 1 kali kirim |

### Leadership

Bahkan tanpa jabatan manager, kamu bisa pimpin:

- **Lead by example** — code quality, deadline, attitude
- **Knowledge sharing** — tech talk internal, documentation
- **Mentoring** — bantu junior naik level
- **Initiative** — usul improvement, fix process yang broken

### Mentoring

**Framework mentoring:**

```
1. Observe — lihat cara kerja mentee
2. Guide — kasih arah, bukan jawaban
3. Review — feedback konstruktif
4. Empower — kasih tanggung jawab
5. Celebrate — akui pencapaian mereka
```

**Mentoring is not:**
- Memberi jawaban langsung
- Mengerjakan tugas mereka
- Micromanaging

### Decision Making

Framework untuk tech decisions:

```
1. Define problem — apa yang mau dipecahkan?
2. Gather data — ukur, riset, benchmarking
3. Options — minimal 2 alternatif
4. Trade-offs — pro/kontra tiap opsi
5. Decision — pilih yang terbaik untuk konteks
6. Document — tulis ADR (Architecture Decision Record)
7. Review — monitor hasilnya
```

### Networking

**Bukan:** minta tolong, minta kerja, minta referral

**Adalah:** bantu orang lain dulu, bagi ilmu, tanya kabar

**Cara networking efektif:**
1. Hadiri meetup / conference — kenalan 3-5 orang baru
2. LinkedIn — comment meaningful, bukan asal "Great post!"
3. Komunitas — aktif diskusi, bantu jawab pertanyaan
4. Open source — review PR orang, respon issue
5. Follow-up — kirim pesan setelah ketemu: *"Senang kenalan tadi di [event]"*

### Growth Mindset

| Fixed Mindset | Growth Mindset |
|---------------|----------------|
| "Saya bukan orang yang berbakat coding" | "Saya belum mahir coding — perlu latihan" |
| "Saya terlalu muda/tua" | "Usia bukan hambatan untuk belajar" |
| "Saya takut salah" | "Kesalahan adalah pelajaran" |
| "Dia pinter bawaan" | "Dia kerja keras untuk sampai sana" |
| "Saya sudah mentok" | "Saya belum mencapai potensi maksimal" |

## 6. Latihan Praktik (5 menit)

### Latihan 1: Bikin Career Roadmap 5 Tahun

Buat dokumen roadmap karier pribadi dengan template yang disediakan. Sertakan:

1. **Current state assessment:**
   - Skill saat ini (jujur)
   - Gap analysis dengan target posisi
2. **Milestone per tahun:**
   - Skill yang harus dikuasai
   - Project / sertifikasi
   - Target gaji
   - Target jabatan
3. **Action plan:**
   - Langkah konkret bulan depan
   - Resources (kursus, buku, mentor)
4. **Contingency plan:**
   - Apa yang dilakukan jika stuck?
   - Track alternatif (management vs IC)

### Latihan 2: Roleplay Negosiasi Gaji

Bersama partner:
1. Satu orang jadi recruiter dengan budget Rp 12-15 juta
2. Satu orang jadi kandidat dengan minimal Rp 14 juta
3. Latihan: tanya budget, terima tawaran, counter, trade-off
4. Tukar peran

### Latihan 3: Salary Research

1. Cari 3 lowongan yang sesuai dengan profil kamu
2. Catat range gaji (dari postingan, Glassdoor, atau komunitas)
3. Hitung rata-rata pasar
4. Tentukan MA/MI/MS kamu
5. Siapkan script negosiasi
