# Quiz: Frontend Frameworks

<div class="quiz">

**1. Apa konsep utama dari component-based architecture di frontend framework?**

- [ ] Semua kode dalam satu file HTML
- [x] UI dibagi menjadi komponen kecil reusable yang punya logic dan styling sendiri
- [ ] Setiap halaman adalah file terpisah
- [ ] Komponen hanya bisa dipakai sekali

**2. Framework mana yang menggunakan Virtual DOM untuk optimasi rendering?**

- [ ] Svelte
- [x] React
- [ ] jQuery
- [ ] Bootstrap

**3. Apa fungsi dari `useState` di React?**

- [ ] Menyimpan data di database
- [x] Mendeklarasikan state lokal dalam functional component yang memicu re-render saat berubah
- [ ] Membuat component baru
- [ ] Mengubah URL

**4. Perbedaan utama SSR (Server-Side Rendering) dengan CSR (Client-Side Rendering)?**

- [ ] SSR lebih lambat dari CSR
- [x] SSR merender HTML di server, CSR merender di browser menggunakan JavaScript
- [ ] CSR tidak perlu JavaScript
- [ ] SSR hanya untuk e-commerce

**5. Framework apa yang merupakan meta-framework React untuk SSR/SSG?**

- [ ] Vue
- [x] Next.js
- [ ] Angular
- [ ] SvelteKit

**6. Konsep one-way data binding berarti?**

- [ ] Data mengalir dua arah antara model dan view
- [ ] Data tidak bisa diubah
- [x] Data mengalir dari parent ke child (top-down) melalui props
- [ ] Data hanya bisa diubah oleh user

**7. Library state management global yang populer di ekosistem React?**

- [ ] Vuex
- [x] Redux / Zustand
- [ ] NgRx
- [ ] MobX

**8. Apa itu Static Site Generation (SSG)?**

- [ ] Halaman dirender saat user request
- [x] Halaman HTML di-generate saat build time, siap serve static
- [ ] Halaman menggunakan JavaScript di client
- [ ] Halaman tidak bisa di-index SEO

**9. Directive `v-if` di Vue.js digunakan untuk?**

- [ ] Looping array
- [ ] Binding data ke input
- [x] Conditional rendering (render elemen jika kondisi true)
- [ ] Event handling

**10. Apa keuntungan utama menggunakan TypeScript dengan frontend framework?**

- [ ] Ukuran bundle lebih kecil
- [x] Type checking saat development mengurangi bug runtime
- [ ] Waktu build lebih cepat
- [ ] Tidak perlu testing
