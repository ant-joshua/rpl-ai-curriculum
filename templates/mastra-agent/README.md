# Mastra AI Agent Starter

```bash
npx degit rpl-ai-curriculum/templates/mastra-agent my-agent
cd my-agent
npm install
npm run dev
```

## Structure

```
src/mastra/
  index.ts       # Mastra instance
  agents/        # Agent definitions
  tools/         # Custom tools
```

## Commands

```bash
npm run dev      # Start Mastra Studio (port 4111)
npm run build   # Build
```

## Env Variables

```env
OPENAI_API_KEY=sk-...
ANTHROPIC_API_KEY=sk-ant-...
```
