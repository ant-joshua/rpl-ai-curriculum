# 🛠️ Starter Templates

Template tinggal clone & run. Udah include setup TypeScript, linting, dll.

| Template | Stack | Untuk |
|----------|-------|-------|
| [Basic TypeScript](basic-ts/) | TypeScript 5, ESLint | Latihan TS/JS dasar |
| [Express API](express-api/) | Express + TS + Routing | Backend API |
| [Mastra Agent](mastra-agent/) | Mastra + Agents + Tools | AI agent development |

### Cara Pake

```bash
# Clone template yang lo mau
npx degit path/ke/template my-project
cd my-project
npm install
npm run dev
```

Atau copy manual filenya.
