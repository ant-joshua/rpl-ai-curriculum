# Basic TypeScript Starter

```bash
npx degit rpl-ai-curriculum/templates/basic-ts my-project
cd my-project
npm install
npm run dev
```

## Includes
- TypeScript 5
- ESLint + Prettier
- ts-node-dev (auto restart)
- .env support

## Commands

```bash
npm run dev    # Development with hot reload
npm run build  # Build for production
npm start      # Run compiled code
npm run lint   # Check code quality
```
