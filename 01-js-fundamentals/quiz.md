# Quiz: JavaScript Fundamentals

**1. Keyword untuk mendeklarasikan variabel yang tidak bisa di-reassign?**

- [ ] var
- [ ] let
- [x] const
- [ ] static

**2. Tipe data apa yang dihasilkan oleh `typeof "Hello"`?**

- [ ] object
- [x] string
- [ ] boolean
- [ ] number

**3. Function `map()` pada array digunakan untuk...**

- [ ] Mencari elemen tertentu
- [x] Mentransformasi setiap elemen ke bentuk baru
- [ ] Menyaring elemen berdasarkan kondisi
- [ ] Menggabungkan dua array

**4. Perbedaan `==` dan `===`?**

- [ ] Sama saja
- [x] `==` loose equality (type coercion), `===` strict equality (no coercion)
- [ ] `==` untuk angka, `===` untuk string
- [ ] `===` lebih lambat

**5. Apa output dari `Promise.resolve(1).then(x => x + 1)`?**

- [ ] 1
- [x] Promise yang resolve dengan nilai 2
- [ ] Error
- [ ] undefined

**6. Spread operator `...` pada array digunakan untuk...**

- [ ] Menghapus elemen
- [x] Menyalin atau menyebar elemen array
- [ ] Mengurutkan array
- [ ] Membalik array

**7. `const fruits = ["apel", "mangga"]; fruits.push("jeruk")` — apakah ini error?**

- [ ] Ya, karena const
- [x] Tidak, const hanya melarang reassignment, bukan mutasi
- [ ] Ya, karena push tidak ada di array const
- [ ] Tidak, tapi fruits jadi undefined

**8. `async function` selalu mengembalikan...**

- [ ] Object
- [x] Promise
- [ ] Array
- [ ] Callback

**9. Manakah contoh arrow function yang benar?**

- [ ] function => (x) { return x }
- [x] `const add = (a, b) => a + b`
- [ ] `const add = a, b -> a + b`
- [ ] `const add = (a b) => { return a + b }`

**10. Method array untuk mengiterasi setiap elemen tanpa mengembalikan array baru?**

- [ ] map()
- [x] forEach()
- [ ] filter()
- [ ] reduce()
