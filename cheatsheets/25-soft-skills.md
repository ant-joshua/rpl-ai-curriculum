# 25. 🤝 Soft Skills

## Komunikasi Efektif
| Situasi | Lakukan | Hindari |
|---------|---------|---------|
| Melapor masalah | Fakta + dampak + usul solusi | Menyalahkan |
| Minta bantuan | "Saya stuck di X, sudah coba A,B,C" | "Tolong kerjain" |
| Code review | Spesifik: "fungsi X rawan null" | "Kode kamu jelek" |
| Daily standup | Apa kemarin, hari ini, hambatan | Cerita panjang |

## Teamwork
- **Git discipline**: branch sendiri, commit kecil, PR deskriptif
- **Shared ownership**: semua punya tanggung jawab atas kode
- **Knowledge sharing**: dokumentasi, pairing, tech talk
- **Conflict resolution**: fokus ke masalah, bukan orang

## Time Management
- **Pomodoro**: 25 menit kerja → 5 menit istirahat
- **Prioritas**: Eisenhower Matrix (urgent/important)
- **Timeboxing**: batasi waktu per task (e.g. 1 jam)
- **Task breakdown**: pecah besar jadi kecil (< 2 jam)

## Growth Mindset
```
"I don't understand this" → "I haven't learned this yet"
"I failed" → "I found a way that doesn't work"
"They're smarter" → "I can learn from them"
```

## Menerima Feedback
1. Dengarkan tanpa defensif
2. Klarifikasi jika perlu
3. Ucapkan terima kasih
4. Evaluasi — apa yang bisa diperbaiki?

## Presentasi Teknis
- **Struktur**: Problem → Solusi → Demo → Kesimpulan
- **Slide**: minimal teks, banyak visual
- **Demo**: punya fallback (screenshot/video)
- **Q&A**: "Saya cari tahu dulu" > jawab asal

## Common Pitfalls
- ❌ Over-commit — bilang "tidak" itu profesional
- ❌ Sembunyikan masalah — makin besar
- ❌ Multitasking — fokus satu task dulu
