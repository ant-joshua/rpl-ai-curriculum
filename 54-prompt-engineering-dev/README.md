# Prompt Engineering untuk Developer

![Prompt Engineering](https://images.pexels.com/photos/546819/pexels-photo-546819.jpeg)

> **Level:** Intermediate
> **Prerequisite:** Modul 1 — Dasar AI untuk Developer
> **Output:** Prompt library + Cheat sheet

## Deskripsi

Modul ini mengajarkan developer cara merancang prompt efektif untuk LLM dalam konteks software engineering. Fokus pada teknik praktis: system prompt design, code generation prompting, prompt patterns untuk task developer sehari-hari, dan strategi production-grade prompting.

## Tujuan Pembelajaran

Setelah modul ini, peserta mampu:
- Merancang system prompt untuk berbagai peran engineering
- Menggunakan few-shot, chain-of-thought, dan structured output
- Menerapkan 8 prompt pattern untuk task developer
- Mengelola prompt versioning, testing, dan monitoring di production
