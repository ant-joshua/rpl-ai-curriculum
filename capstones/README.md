# 🏆 Capstone Projects — RPL AI Curriculum

> Project besar yang ngetes semua skill: frontend, backend, database, AI agent, deployment.
> Estimasi: **4-6 minggu** per capstone (bisa dikerjain berkelompok 2-3 orang).

## Cara Pake

1. Pilih 1 capstone yang paling menarik minat
2. Ikuti sprint plan — tiap sprint 1-2 minggu
3. Presentasi final + demo di akhir

## Sprint Structure (4 sprint × 2 minggu)

| Sprint | Fokus | Output |
|--------|-------|--------|
| Sprint 1 | Foundation: setup, database, auth | Schema, API dasar, login/register |
| Sprint 2 | Core Features: CRUD utama | Fitur inti jalan, frontend dasar |
| Sprint 3 | AI Integration: Mastra agent + tools | AI features functional |
| Sprint 4 | Polish: testing, deploy, presentasi | Live di Vercel+Railway, presentasi |

## Penilaian Capstone

| Kriteria | Bobot |
|----------|-------|
| Semua fitur jalan | 30% |
| Frontend rapi + responsive | 20% |
| Backend + database solid | 20% |
| AI feature integrated | 15% |
| Deployed + demo | 10% |
| Code quality + Git | 5% |

## Deliverables Wajib

- [ ] Repo GitHub dengan README rapi
- [ ] Deployed (frontend Vercel, backend Railway)
- [ ] Demo video (3-5 menit)
- [ ] Slide presentasi (5 slide max)
- [ ] AI Prompt Log (PROMPT-LOG.md)
