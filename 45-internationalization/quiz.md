# Quiz: Internasionalisasi (i18n)

> Jawab pertanyaan di bawah dengan klik pilihan yang benar.

<div class="quiz">

**1. Perbedaan utama antara i18n (internationalization) dan l10n (localization)?**

- [ ] Sama saja, hanya singkatan berbeda
- [x] i18n = menyiapkan kode agar bisa diadaptasi; l10n = menerjemahkan konten ke bahasa/region tertentu
- [ ] i18n = untuk web; l10n = untuk mobile
- [ ] i18n = frontend; l10n = backend

**2. Format ICU Message digunakan untuk menangani...**

- [ ] Styling komponen
- [x] Pluralisasi, gender, dan interpolasi dalam terjemahan
- [ ] Routing multi-language
- [ ] CSS logical properties

**3. Perbedaan utama RTL (Right-to-Left) dan LTR di aplikasi web?**

- [ ] RTL hanya mengubah bahasa, layout tetap sama
- [x] RTL membutuhkan CSS logical properties dan mirroring layout (arah teks, margin, padding)
- [ ] RTL tidak bisa digabung dengan React
- [ ] RTL hanya untuk bahasa Arab dan Ibrani

**4. Library React yang paling umum untuk i18n?**

- [ ] react-intl
- [x] react-i18next
- [ ] react-i18nify
- [ ] react-locale

**5. Untuk mendeteksi bahasa pengguna di sisi server Express, header HTTP yang digunakan?**

- [ ] Content-Language
- [x] Accept-Language
- [ ] X-Locale
- [ ] Language-Preference

**6. Alur kerja terjemahan (translation workflow) production yang benar?**

- [ ] Developer menulis kode → Product Manager menerjemahkan → Deploy
- [ ] Deploy dulu → Terjemah belakangan → Patch
- [x] Developer extract key → Translator translate → Reviewer review → QA test → Deploy
- [ ] Translator tulis langsung di source code → Developer review
