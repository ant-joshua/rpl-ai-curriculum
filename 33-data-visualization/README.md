# 33. Data Visualization

> **Level:** 🌱 Beginner → 🌿 Intermediate
> **Jam:** 8 (4 minggu × 2 sesi)
> **Prasyarat:** JavaScript fundamentals, HTML/CSS, basic DOM manipulation
> **Output:** Interactive data dashboard — real-time charts, multiple chart types, D3.js custom viz

## Tujuan Pembelajaran

Setelah modul ini, kamu bisa:

- Bedain chart types (line, bar, pie, scatter, area) dan kapan pake masing-masing
- Setup & konfigurasi Chart.js 4 — datasets, options, responsive, animations
- Bikin mixed charts, time series, zoom/pan, annotations, custom tooltips
- Export chart sebagai image
- Paham D3.js core concepts: select, data join, scale, axis, SVG
- Bikin bar chart & line chart dari scratch pake D3.js
- Transisi & animasi data di D3.js
- Integrasi real-time data via WebSocket + Chart.js
- Bikin dashboard layout pake CSS Grid — dark mode, filtering, date range
- Sinkronisasi multiple chart di dashboard
