**1. Fitur GitHub yang menampilkan README di halaman profil pengguna?**

- [x] Repository dengan nama yang sama dengan username
- [ ] Repository pinned
- [ ] Gist README

**2. Platform paling umum untuk mendeploy portfolio website statis gratis?**

- [ ] AWS EC2
- [x] Vercel / Netlify
- [ ] Google Cloud Run

**3. Bagian profil LinkedIn yang paling berpengaruh untuk muncul di pencarian rekruter?**

- [ ] Foto profil
- [x] Headline (judul profesional)
- [ ] Background foto
- [ ] Bagian volunteer

**4. Format CV yang paling disarankan agar lolos ATS (Applicant Tracking System)?**

- [ ] PDF dengan desain grafis rumit
- [x] PDF sederhana berbasis teks tanpa kolom/layout kompleks
- [ ] Docx dengan gambar dan tabel
- [ ] HTML dengan CSS kustom

**5. Nama perintah untuk memberi anotasi kode dengan kontributor eksternal di GitHub?**

- [ ] Fork
- [ ] Star
- [x] Pull Request
- [ ] Issue

**6. Platform blog teknis yang paling populer di kalangan developer untuk menulis artikel pemrograman?**

- [ ] WordPress
- [ ] Medium
- [x] Dev.to / Hashnode
