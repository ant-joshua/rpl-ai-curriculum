# Modul 44: Digital Portfolio & Personal Branding

![Banner Portfolio & Branding](https://images.pexels.com/photos/1181263/pexels-photo-1181263.jpeg)

> Bangun personal branding digital yang kuat — dari portofolio website, profil GitHub, LinkedIn, hingga kontribusi komunitas.

## Informasi Modul

| Item | Detail |
|------|--------|
| **Tingkat** | Mahir (Intermediate) |
| **Durasi** | 16 Jam (4 sesi × 4 jam) |
| **Prasyarat** | Lulus modul dasar web (HTML, CSS, JavaScript), akun GitHub & LinkedIn aktif |
| **Tools** | VS Code, GitHub Pages, Vercel/Netlify, Shields.io, Canva/Figma (opsional) |

## Tujuan Pembelajaran

Setelah menyelesaikan modul ini, peserta mampu:

1. Membuat GitHub Profile README profesional dengan badge, pin repositori, dan dokumentasi proyek
2. Membangun dan mendeploy portfolio website responsif dengan studi kasus proyek
3. Mengoptimasi profil LinkedIn dan menyusun CV/Resume ATS-friendly
4. Menulis artikel teknis dan berkontribusi di komunitas open source

## Materi Pembelajaran

| Sesi | Topik | File | Durasi | Output |
|------|-------|------|--------|--------|
| 1 | **GitHub Profesional** — Profile README, pinned repos, badge shields.io, dokumentasi proyek, GitHub Pages | [view](01-github-profile.md) | 4 jam | GitHub profile lengkap + project docs |
| 2 | **Portfolio Website** — Hero, about, project cards, skills, contact form, case study pages, deploy Vercel/Netlify | [view](02-portfolio-website.md) | 4 jam | Portfolio website live |
| 3 | **LinkedIn & Career** — Optimasi profil, headline, about, featured projects, CV ATS, personal branding statement | [view](03-linkedin-career.md) | 4 jam | Profil LinkedIn optimal + CV siap pakai |
| 4 | **Tech Community** — Blog teknis (Dev.to/Hashnode), artikel problem-solution, open source PR, networking | [view](04-tech-community.md) | 4 jam | Artikel blog + PR open source |

## Output Akhir

Setelah menyelesaikan seluruh sesi, peserta menghasilkan:

- ✅ **Live Portfolio Website** — di-deploy ke Vercel/Netlify dengan domain kustom
- ✅ **GitHub Profile Profesional** — profile README, pinned repos, badges, GitHub Pages site
- ✅ **Profil LinkedIn Optimal** — headline, about, featured, skills endorsement
- ✅ **CV/Resume ATS-Friendly** — siap dikirim ke perekrut
- ✅ **Artikel Teknis Publik** — di Dev.to / Hashnode / Medium
- ✅ **Kontribusi Open Source** — minimal 1 commit/PR ke repositori publik
